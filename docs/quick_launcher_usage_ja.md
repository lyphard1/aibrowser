# ワンクリック起動（ターミナル入力不要）

`launchers` フォルダに、ダブルクリック用の `.command` を作成済みです。

- `launchers/AI_Start.command`  
  managed 5窓を起動・整列（必要なら extra 復元）
- `launchers/AI_Relayout.command`  
  managed 5窓だけ再整列
- `launchers/AI_Close.command`  
  extra 保存 + managed クローズ

## 使い方
1. Finder で `launchers` フォルダを開く  
2. 目的の `.command` をダブルクリック

## 最初だけ必要な場合があること
macOS が「開発元未確認」と出したら、右クリック → 「開く」で許可してください。

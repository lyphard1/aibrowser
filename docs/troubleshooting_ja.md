トラブルシュート
================

1) 再整列しても位置が合わない
----------------------------

`scripts/ai_window_tool.sh` の `LEFT_BOUNDS` / `RIGHT_BOUNDS` を実モニターに合わせる。

例（2560x1440 x2）:
- LEFT_BOUNDS="0,30,2560,1410"
- RIGHT_BOUNDS="2560,30,5120,1410"

2窓側の幅調整は `GROUP_B_WINDOW_SCALE` で行える（`1.2` = 20%広い）。

2) 一部ウィンドウだけ動かない
---------------------------

再整列対象は `PROVIDER_MATCHES` に一致するウィンドウだけ。
対象外URLにいると動かない。

3) 生成AI以外のSafariウィンドウを触ってしまうのが怖い
-----------------------------------------

このツールはURL判定でAI窓だけを対象化しているので、
通常Webウィンドウは移動しない。

4) Hammerspoon自動固定が強すぎる
-----------------------------

`hammerspoon/ai_window_lock.lua` の `AUTO_LOCK` を `false` に戻す。
まずは手動ホットキーだけで運用するのがおすすめ。

5) 権限ダイアログが出る
----------------------

macOSの「アクセシビリティ」「自動操作」権限で、
Terminal/Hammerspoon/Safari を許可する。
